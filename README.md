# Weekly Recipe Planner & Smart Shopping List

A full-stack application built with **Rust (Axum)** and **React**.

## Features

- **Recipe Management**: Create and list recipes with ingredients.
- **Weekly Planning**: Add recipes to your weekly meal plan.
- **Smart Shopping List**: Automatically generate a shopping list with aggregated ingredients and estimated costs.

## Prerequisites

- **Rust**: [Install Rust](https://www.rust-lang.org/tools/install)
- **Node.js**: [Install Node.js](https://nodejs.org/)

## Getting Started

### Backend

1.  Navigate to the backend directory:
    ```bash
    cd backend
    ```
2.  Run the server:
    ```bash
    cargo run
    ```
    The server will start at `http://localhost:3000`.

### Frontend

1.  Navigate to the frontend directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:5173` (or similar).
